# teco

## Usage

```sh
Usage: teco --target <TARGET> --case-dir <CASE_DIR>

Options:
  -t, --target <TARGET>      타겟 바이너리 경로
  -c, --case-dir <CASE_DIR>  *.in과 *.out 파일이 있는 디렉토리
  -h, --help                 Print help
  -V, --version              Print version
```


- [ ] target에 arguments를 넣을 수 있도록 수정
- [ ] input, output 파일을 지정할 수 있도록 수정
